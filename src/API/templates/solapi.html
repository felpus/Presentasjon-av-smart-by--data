The SOL restAPI

Functions for all sensors

URL Description Parameters (Required in bold) Example
/solapi/sensors Returns a list of all sensors in the database and their information. None /solapi/sensors
/solapi/recent/latest Returns the latest reading of a specified sensor by sensorid. sensorid /solapi/recent/latest?sensorid=00000466
/solapi/rawdata/date Returns the raw ungrouped data from a sensor in the database. The sensor is specified by sensorid, and requires at least a specific year as parameter. Can also return by month and day. sensorid, year, month, day /solapi/rawdata/date?sensorid=00000466&year=2020&month=6&day=1
/solapi/rawdata/timeperiod Returns the raw ungrouped data from a sensor in the database. The sensor is specified by sensorid, and the time period is between the two specified dates in a YYYY-MM-DD format. sensorid, startdate, enddate /solapi/rawdata/timeperiod?sensorid=00000466&startdate=2020-5-20&enddate=2020-6-20

Sensor specific functions

TYPE Description
watertemp Only available for sensors that contains the "temperature_b" field.
airtemp Only available for sensors that contains the "temperature" field.
ph Only available for sensors that contains the "ph_a" field.
upcoming More types will be added as new sensors are added to the system.

URL Description Parameters (Required in bold) Example
/solapi/<type>/recent/day Returns the averaged data of a specific sensor from the last 24 hours, groups the data by the hour. sensorid solapi/watertemp/recent/day?sensorid=00000466
/solapi/<type>/recent/week Returns the averaged data of a specific sensor from the last week, groups the data by the hour. sensorid solapi/watertemp/recent/week?sensorid=00000466
/solapi/<type>/recent/month Returns the averaged data of a specific sensor from the last month, groups the data by the day. sensorid solapi/watertemp/recent/month?sensorid=00000466
/solapi/<type>/recent/year Returns the averaged data of a specific sensor from the last year, groups the data by the month. sensorid solapi/watertemp/recent/year?sensorid=00000466
/solapi/<type>/date Returns the averaged data of a specific sensor from the specified date. Selecting an entire year groups the data by month. Selecting by year and month groups the data by day, and selecting by year, month and day groups the data by hour. sensorid, year, month, day solapi/watertemp/date?sensorid=00000466&year=2020&month=6&day=1
/solapi/<type>/timeperiod Returns the averaged data of a specific sensor from the specified period. The sensor is specified by sensorid, and the time period is between the two specified dates in a YYYY-MM-DD format. Also allows for grouping of data by year, month, week, day and hour. sensorid, startdate, enddate, groupby solapi/watertemp/timeperiod?sensorid=00000466&startdate=2020-05-1&enddate=2020-06-1&groupby=day
